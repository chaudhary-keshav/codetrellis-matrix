**Problem:** Position interface appearing twice in matrix.prompt output.

**Solution:** Added deduplication methods to `MatrixCompressor`:

```python
def _deduplicate_interfaces(self, interfaces: List[Dict]) -> List[Dict]:
    """Remove duplicate interfaces based on name and file_path."""
    seen = {}
    for iface in interfaces:
        name = iface.get('name', '')
        file_path = iface.get('file_path', iface.get('filePath', ''))
        key = (name, file_path)
        if key not in seen:
            seen[key] = iface
    return list(seen.values())
```

## 🔍 Key Decisions Made

### 1. Version Selection

**Decision:** Use 4.1.0 as the unified version
**Rationale:** Cache folder already shows 4.1.0, indicating this is the latest development state

### 2. Deduplication Strategy

**Decision:** Deduplicate by (name, file_path) tuple
**Rationale:** Same interface name in different files should be kept, only exact duplicates removed

### 3. Error Isolation Approach

**Decision:** Use ExtractorResult wrapper + ErrorCollector pattern
**Rationale:**

- Allows extraction to continue even when files fail
- Collects all errors for reporting at end
- Decorator makes adoption easy

### 4. Error Code Format

**Decision:** EXXX format with category prefixes
**Rationale:**

- Easy to identify error category (E1XX = file, E2XX = parse, etc.)
- Room for expansion within categories
- Machine-readable for tooling
